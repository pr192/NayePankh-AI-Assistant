# 🌱 NayePankh AI Assistant

An interactive web-based assistant designed for **NayePankh Foundation** to help visitors quickly access information about internships, volunteering opportunities, donations, certificates, and community initiatives.

## 🚀 Project Overview

NayePankh AI Assistant aims to improve user engagement by providing a simple and user-friendly chatbot experience. The assistant helps visitors navigate the foundation's activities and discover opportunities to contribute to social impact initiatives.

This project was developed as part of the **AI Web Development Internship Assessment**.

---

## ✨ Features

### 🤖 Interactive Chatbot

* Answers common questions related to:

  * Internships
  * Volunteering
  * Donations
  * Certificates
  * Events
  * Foundation Information

### 🌙 Dark Mode

* Modern dark/light theme toggle
* Improved accessibility and user experience

### ⚡ Quick Action Buttons

* Volunteer
* Donation
* Certificate

### ⌨️ Typing Animation

* Simulates a real assistant interaction
